# Secure Profile API

REST API sederhana yang dikembangkan menggunakan Node.js dan Express sebagai tugas Mata Kuliah **Pemrograman Web II**.

## Fitur
- Registrasi pengguna
- Login dengan JWT Authentication
- Validasi input menggunakan Express Validator
- Password hashing menggunakan bcryptjs
- Protected route dengan Bearer Token
- Rate limiting dan error handling

## Teknologi
- Node.js
- Express.js
- JWT (jsonwebtoken)
- bcryptjs
- express-validator
- dotenv
